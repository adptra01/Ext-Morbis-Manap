 Panduan Penggunaan


                    

                        

                            

                                
                            

                            

                                

                                    
    
                                        
  1. Pastikan Chrome Extension sudah terinstall di browser
    2. 
                                        
  2. Buka halaman M-KLAIM: http://103.147.236.140/v2/m-klaim
    3. 
                                        
  3. Login dengan akun Anda
    4. 
                                        
  4. Buka tool ini di tab terpisah
    5. 
                                    

                                

                            

                        


                        

                            

                                
                            

                            

                                

                                    
    
                                        
  1. Isi form URL Builder dengan parameter yang diinginkan
    2. 
                                        
  2. Klik tombol "Generate URL"
    3. 
                                        
  3. Copy URL yang dihasilkan
    4. 
                                        
  4. Buka URL di browser baru
    5. 
                                        
  5. Halaman akan terbuka dengan filter yang benar
    6. 
                                    

                                

                            

                        


                        

                            

                                
                            

                            

                                

                                    
    
                                        
  1. Buka tab "Bookmarklet" di tool ini
    2. 
                                        
  2. Drag tombol "Fix M-KLAIM" ke bookmarks bar browser Anda
    3. 
                                        
  3. Buka halaman M-KLAIM (list atau detail)
    4. 
                                        
  4. Klik bookmarklet yang sudah disimpan
    5. 
                                        
  5. Popup konfirmasi akan muncul - klik OK
    6. 
                                        
  6. Fungsi navigasi sekarang sudah diperbaiki!
    7. 
                                    

                                    

                                        
                                        Catatan: Bookmarklet harus diklik setiap kali halaman M-KLAIM dimuat ulang.
                                    

                                

                            

                        


                        

                            

                                
                            

                            

                                

                                    
    
                                        
  1. Buka tab "Testing" di tool ini
    2. 
                                        
  2. Klik tombol "Jalankan Testing"
    3. 
                                        
  3. Tool akan mengecek status semua fitur extension
    4. 
                                        
  4. Hasil test akan ditampilkan di bawah tombol
    5. 
                                        
  5. Copy hasil test jika perlu untuk pelaporan
    6. 
                                    

                                

                            

                        


                        

                            

                                
                            

                            

                                

                                    

                                        
                                            

                                                Masalah
                                                Solusi
                                            

                                        
                                        
                                            

                                                Tombol Kembali reset filter
                                                Gunakan Bookmarklet Fix
                                            

                                            

                                                Extension tidak merespon
                                                Refresh halaman atau restart browser
                                            

                                            

                                                URL error setelah verifikasi
                                                Gunakan Bookmarklet Fix sebelum verifikasi
                                            

                                            

                                                Bookmarklet tidak bisa didrag
                                                Copy kode bookmarklet dan buat bookmark manual
